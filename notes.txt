برای اضافه کردن کد خود به GitHub، می‌توانید مراحل زیر را دنبال کنید:

1. ایجاد یک مخزن (repository) در GitHub: وارد حساب کاربری خود در GitHub شوید و یک مخزن جدید بسازید. به عنوان نمونه، نام مخزن را مشخص کنید و گزینه "Initialize this repository with a README" را فعال کنید.

2. متصل شدن به Git در محیط خط فرمان: برای اتصال به مخزن GitHub خود، نیاز به استفاده از Git دارید. در محیط خط فرمان (Command Line Interface) خود، به مسیر مورد نظر بروید و دستورات زیر را وارد کنید:

```
git init
git remote add origin <remote_repository_URL>
```

در این دستور، `<remote_repository_URL>` را با URL مخزن GitHub خود جایگزین کنید.

3. اضافه کردن فایل‌ها: فایل‌های مورد نظر خود را به مخزن اضافه کنید. برای این کار، دستور زیر را وارد کنید:

```
git add .
```

این دستور تمام فایل‌های تغییر یافته در دایرکتوری فعلی را به مخزن اضافه می‌کند.

4. تأیید تغییرات: تغییرات اعمال شده را تأیید کنید. از دستور زیر استفاده کنید:

```
git commit -m "Commit message"
```

در جای "Commit message"، یک پیام مختصری برای توضیح تغییرات خود بنویسید.

5. Push تغییرات به مخزن GitHub: تغییرات را به مخزن GitHub خود push کنید. از دستور زیر استفاده کنید:

```
git push origin main
```

در اینجا، "main" باید با نام شاخه پیش فرض شما در مخزن GitHub جایگزین شود.

پس از انجام این مراحل، کد شما به مخزن GitHub شما اضافه خواهد شد. در صفحه مخزن خود در GitHub می‌توانید تغییرات را مشاهده کنید و آن را به اشتراک بگذارید.
